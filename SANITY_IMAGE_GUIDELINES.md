# Sanity Image Guidelines

- Use Sanity image fields (no Cloudinary). Images are stored in Sanity's asset store.
- Provide alt text for accessibility and SEO.
- Recommended sizes:
  - Hero / main image: 1600x900 (or wider)
  - Thumbnails: 400x300
- Use `hotspot` for focal point control in the editor (enabled on most image fields).
