## Before final domain switch

* backup WordPress
* export WordPress URLs
* confirm redirect map
* verify new Firebase site
* verify forms
* verify Sanity connection
* verify sitemap
* verify robots.txt
* verify mobile layout
* verify key pages
* verify email/MX records will not be touched
* prepare DNS plan

## During final domain switch

* do not touch MX/email records
* only update web DNS records needed for Firebase Hosting
* verify Firebase custom domain setup
* verify SSL certificate
* verify `www` and apex domain
* verify redirects
* test key pages

## After final domain switch

* submit sitemap to Google Search Console
* inspect homepage URL
* inspect key service URLs
* monitor 404s
* monitor indexing
* monitor redirects
* keep WordPress backup
* migrate remaining blog content in batches
