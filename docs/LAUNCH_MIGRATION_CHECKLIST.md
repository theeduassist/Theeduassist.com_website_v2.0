# Launch Migration Checklist

This checklist outlines the critical steps required before, during, and after the launch of the new website to ensure a smooth transition from WordPress.

## Before launch

*   [ ] Full WordPress backup
*   [ ] Full cPanel backup
*   [ ] Database export
*   [ ] Media export
*   [ ] DNS export
*   [ ] Email DNS records documented
*   [ ] Top URLs exported from Search Console
*   [ ] Redirect map reviewed
*   [ ] Forms tested
*   [ ] Calendly tested
*   [ ] Sitemap generated
*   [ ] robots.txt checked
*   [ ] Analytics env vars added if ready
*   [ ] Cloudflare/Firebase preview tested

## During launch

*   [ ] Deploy final build
*   [ ] Check homepage
*   [ ] Check main service pages
*   [ ] Check old URLs redirect
*   [ ] Check contact form
*   [ ] Check Calendly
*   [ ] Check Outlook email send/receive
*   [ ] Check SSL
*   [ ] Check www and non-www
*   [ ] Check mobile

## After launch

*   [ ] Submit sitemap in Google Search Console
*   [ ] Submit sitemap in Bing Webmaster Tools
*   [ ] Inspect main URLs
*   [ ] Monitor 404s
*   [ ] Monitor form submissions
*   [ ] Monitor analytics
*   [ ] Keep WordPress/GoDaddy backup for 30–60 days
*   [ ] Do not cancel GoDaddy immediately if email depends on cPanel
