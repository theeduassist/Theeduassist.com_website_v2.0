# Blog Migration Pending

This document details the strategy and status of the blog migration from WordPress to the new website.

*   **Status:** Blog migration is not complete yet.
*   **Redirect Rules:** Do not add fake blog redirects.
*   **Prioritization (Traffic):** First migrate top Search Console traffic posts.
*   **Prioritization (Topics):** First migrate Kajabi, LMS, AI/eLearning, and custom eLearning posts.
*   **Prioritization (SEO):** Posts with backlinks must be prioritized.
*   **Timing:** Blog post redirects will be added once target pages exist.
*   **Cleanup:** Some weak posts may be merged, noindexed, or not migrated.
