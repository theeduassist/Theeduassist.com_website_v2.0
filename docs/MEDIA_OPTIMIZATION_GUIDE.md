# Media Optimization Guide

- Do not bulk import old WordPress media.
- Use descriptive filenames.
- Use alt text for meaningful images.
- Decorative images should have empty alt where appropriate.
- Use SVG for icons/logos.
- Use WebP/AVIF where possible for large images.
- Avoid uploading huge Canva PNGs.
- Compress hero images before use.
- Do not store large videos directly in Sanity.
- Use external video hosting or later storage strategy.
- PDFs/workbooks should be optimized before upload.
- Avoid duplicate images.
- Keep image width appropriate for display.
- Use lazy loading for non-hero images where possible.
- Do not lazy-load the main hero image if it hurts LCP.
- Ensure images do not cause layout shift by setting dimensions where possible.
