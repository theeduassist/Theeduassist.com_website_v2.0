# Logo and Favicon Implementation

## Logo Assets
- `public/brand/theeduassist-logo.svg`: Combined mark and wordmark with tagline
- `public/brand/theeduassist-logo-mark.svg`: Standalone emblem/mark
- `public/brand/theeduassist-logo-wordmark.svg`: Standalone wordmark

## Favicons
- `public/favicon.svg`
- `public/apple-touch-icon.png` (fallback)
- `public/site.webmanifest`

## Usage Guidelines
- The full logo should be used in the header and footer where appropriate.
- The mark alone can be used as a favicon or social icon.
- Fallback text (e.g. `TheEduAssist`) should be used if images fail to load.
