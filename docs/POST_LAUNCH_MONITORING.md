# Post-Launch Monitoring Plan

## First 1 hour

* homepage
* main pages
* SSL
* www/non-www
* forms
* Calendly
* email
* old URL redirects

## First 24 hours

* Cloudflare/Firebase deployment logs
* 404s
* form submissions
* analytics
* Search Console
* Bing
* email deliverability
* page speed

## First 7 days

* Search Console coverage
* ranking/traffic baseline
* 404 review
* redirect fixes
* lead form review
* Clarity recordings if enabled
* Sanity content edits
* performance issues

## First 30–60 days

* Keep GoDaddy/cPanel/WordPress backup
* Migrate priority blog posts
* Add missing redirects
* Review hosting costs
* Review media usage
* Decide whether to keep/downgrade GoDaddy hosting
* Do not cancel email-dependent services until confirmed independent
