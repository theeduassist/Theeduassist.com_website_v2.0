# Content Team Onboarding

This document is for the content team to get started with Sanity Studio.

- Access Studio at `http://localhost:3333/` when running locally.
- Content types available: Pages, Services, Kajabi Services, Case Studies, Blog Posts, FAQs, Authors, Platforms, Training Solutions, Testimonials.
- Add images using the built-in Sanity image field. Use descriptive alt text.
- For SEO, fill out the SEO card on each document.

If you need training, follow the `SANITY_IMAGE_GUIDELINES.md` for image standards.
